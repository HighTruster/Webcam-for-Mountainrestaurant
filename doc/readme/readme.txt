Ein Bergrestaurant gibt uns den Auftrag,
eine Web-Applikation mit Standbildern einer Webcam zu entwickeln.

Das Panorama, welches von diesem Bergrestaurant aus zu sehen ist, ist wunderschön
Daher möchte der Auftraggeber möglichst das ganze Panorama (ca. 180°) zeigen
Dazu soll alle fünf Minuten ein Panoramabild entstehen
welches auf der Hompage angezeigt wird.
Ferner wünscht der Auftraggeber ein Übersichtliches Bildarchiv
mit Panoramabildern der letzen 14 Tage.
Ältere Panoramabilder sollen einmal täglich gelöscht werden.
